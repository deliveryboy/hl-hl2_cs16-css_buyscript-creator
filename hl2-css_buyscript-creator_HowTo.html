Information:

  • Web-Anwendung zum Erstellen eigener BuyScripts für STEAM-CS:S.
  • (Half-Life2 Counter-Strike:Source).
  • Basierend auf JavaScript.
  • Lauffähig auf Mozilla, FireFox, Opera und IE6.
  • Nicht lauffähig auf MacIE, Safari und Konqueror.

Anleitung:

  • Das Formular besteht aus Auswahl-Menüs und einem Text-Feld.
  • Zuerst eine Taste aus dem ersten Auswahl-Menü wählen.
  • Neue Auswahl-Menüs werden erscheinen.
  • Dann eine oder mehrere Zuweisung(en) auswählen.
  • Und so weiter.
  • Eine Übersicht über die verwendbaren Tasten liefert das Bild des Tastatur-Layouts.
  • Die getroffene Auswahl taucht als fertiges BuyScript im Text-Feld darunter auf.
  • Auch kann dort das fertige BuyScript nach Belieben editiert werden.
  • Und zum Schluß kann das BuyScript "kopiert & eingefügt" werden wo auch immer man es haben will.

BUGz:

  • Die Auswahl eines Equipment-Separator ergibt einen leeren Bind.
  • Falls Fehler irgendeiner Art festgestellt werden sollten, bitte diese auf individellem Wege dem Autor mitteilen.
